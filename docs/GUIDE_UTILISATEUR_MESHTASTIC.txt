# Guide utilisateur Meshtastic (sanitizé)

Accès aux outils (à adapter avec vos domaines sécurisés) :
- Interface Node-RED : https://<PUBLIC_DOMAIN>/status (auth requise)
- Grafana : https://<PUBLIC_DOMAIN>/grafana (SSO requis)

Bonnes pratiques :
- Ne jamais partager de PSK ; utiliser uniquement des identifiants de clés (PSK_REF).
- Vérifier que le module est provisionné via le tunnel VPN ou le réseau Wi-Fi de confiance.
- Les URL locales précédentes ont été remplacées par des domaines protégés.
